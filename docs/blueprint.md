# **App Name**: UGO AI Studio

## Core Features:

- AI Chat System: Smart ChatGPT-style conversations with multiple modes (Developer, Designer, Business, Writer). Includes voice input/output, offline-saved chat history with online sync, export options (PDF/PNG/Text), shareable chat links, and cross-device chat continuation via Firebase.
- AI Image Generator: Generate images in multiple styles (realistic, 3D, anime, logo, luxury, etc.). Features image upscaling, background removal, image editing, watermarking, offline image viewing using cache storage, and gallery workspace storage.
- Document & PDF AI Tools: Tools include PDF summarizer, PDF to text conversion, AI explanation of PDF content, PDF to DOCX converter, and AI-generated PDF creation. Results are saved offline (IndexedDB) and auto-synced later.
- Text Intelligence Tools: Features include paraphrasing, grammar correction, text-to-ideas generation, article writer, email writer, social media caption generator, and an offline-ready text editor (like Google Docs).
- Workspace System: A mini Notion built into the app, featuring notes, tasks, AI-generated documents, saved chats, and saved images. All available offline and sync to Firebase when online.
- User Profiles & Cloud Sync: Firebase Authentication (Email, Google, Phone) enables data syncing across devices, offline work with auto-upload, and device installation detection for PWA.
- PWA Core Features: Built-in PWA features including installable app capabilities (Add to Home Screen), offline mode (cached assets, saved content access), push notifications (FCM), background sync, and caching/performance optimizations.
- Universal Share System: Share chats, images, documents, AI workflows, and generate shareable public URLs.
- App-Wide Search: Instantly search anything: chats, images, documents, tools, templates.
- Settings & Personalization: Settings for Light/Dark theme, chat model selection, voice choice, data backup, notification preferences, and AI memory preferences.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey professionalism and trust.
- Background color: Very light blue (#E8EAF6), almost white, to provide a clean and unobtrusive backdrop.
- Accent color: Soft purple (#8E24AA), a visually distinct hue slightly different from blue, to add sophistication.
- Body and headline font: 'Inter', a sans-serif font that provides a modern and neutral look suitable for both headlines and body text.
- Use consistent and modern icons that align with the AI theme. The icons should be simple and easily recognizable.
- Clean and intuitive layout with clear sections for different AI tools and features.
- Subtle transitions and animations to enhance user experience without being distracting.
